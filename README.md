# 🏰 Camelot: Rise of Excalibur – Grand Fair Event

A mobile-first Unity game featuring jousting mini-games, dragon egg hunts, and a medieval Camelot event system.

## 🎮 Core Features
- Jousting mini-game (tap/swipe combat)
- Dragon Egg Hunt collectible system
- Character interactions:
  - King Arthur
  - Guinevere
  - Lancelot
  - Merlin
- Event point progression system
- Reward system (gold, essence, cosmetics)

## 🧱 Tech Stack
- Unity (2D mobile game)
- Node.js backend (optional)
- Express API for events and leaderboards

## 📁 Structure
- `unity-client/` → Game client
- `backend/` → API server
- `docs/` → Game design + economy docs
- `assets/` → Marketing + UI assets

## 🚀 Setup (Unity)
1. Open Unity Hub
2. Add project: `unity-client/`
3. Open scene: `CamelotFair.unity`
4. Press Play
